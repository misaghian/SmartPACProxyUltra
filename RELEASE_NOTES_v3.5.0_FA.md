# Smart PAC Ultra v3.5.0

## تغییرات اصلی

- بازطراحی کامل Popup به مدل فشرده و صفحه‌اصلی‌محور.
- نمایش مستقیم لیست‌های «مستقیم» و «پروکسی» در صفحه اصلی Popup.
- انتقال امکانات اصلی به دسترس‌ترین بخش‌ها: حالت اتصال، سایت فعلی، لیست‌ها، تنظیم سریع پروکسی، تست اتصال و آپدیت.
- افزودن Download Update از GitHub Releases.
- ساخت پشتیبان JSON از تنظیمات قبل از دانلود آپدیت.
- حفظ تنظیمات، پروفایل‌ها و لیست‌ها با Storage Key پایدار.
- افزودن workflow انتشار GitHub Release با فایل ZIP و latest.json.
- اصلاح مسیر فونت VazirFD از `ui/fonts/` و مسیر جایگزین `assets/ui/fonts/`.
- حفظ پشتیبانی قوانین IP مستقیم و پروکسی از نسخه 3.4.2.

## نکته نصب آپدیت

Chrome Extension به دلایل امنیتی نمی‌تواند خودش فایل ZIP را جایگزین و نصب کند. در نسخه GitHub، افزونه فایل آپدیت را دانلود می‌کند و کاربر باید آن را در `chrome://extensions` بارگذاری/Reload کند. در انتشار رسمی Chrome Web Store، آپدیت توسط خود Chrome انجام می‌شود و تنظیمات کاربران حفظ می‌شود.
