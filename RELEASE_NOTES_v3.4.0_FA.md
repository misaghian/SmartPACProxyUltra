# Smart PAC Ultra v3.4.0

این نسخه برای آماده‌سازی انتشار عمومی و Release در GitHub ساخته شده است.

## تغییرات مهم

- اصلاح عبارت‌های رابط کاربری از «دایرکت» به «مستقیم».
- نمایش پایدار نتیجه تست اتصال داخل Popup.
- اضافه شدن نسخه‌بندی داخلی و نمایش نسخه فعلی.
- اضافه شدن بررسی آپدیت از آدرس GitHub Releases.
- اضافه شدن فایل‌های کمکی برای ساخت Release.

## فایل Release پیشنهادی

`smart-pac-ultra-v3.4.0-release.zip`

## نصب تستی

1. فایل را Extract کنید.
2. وارد `chrome://extensions` شوید.
3. Developer Mode را فعال کنید.
4. پوشه `smart-pac-ultra-v3.4.0` را با Load unpacked نصب کنید.
