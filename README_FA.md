# Smart PAC Ultra v3.4.0

افزونه فارسی و حرفه‌ای برای مدیریت حالت‌های اتصال مرورگر: مستقیم، پروکسی، PAC، پروکسی سیستم و هوشمند.

## امکانات

- پروفایل‌های چندگانه پروکسی
- پشتیبانی از HTTP / HTTPS / SOCKS4 / SOCKS5 / MTProto Bridge
- لیست مشترک یا لیست اختصاصی برای هر پروفایل
- پیشنهاد هوشمند هنگام باز نشدن سایت
- تست اتصال پروکسی با نمایش نتیجه داخل Popup
- بررسی آپدیت از GitHub Releases
- Export / Import تنظیمات

## آپدیت

مخزن پیشنهادی پروژه:

```text
https://github.com/misaghian/SmartPACProxyUltra
```

دکمه «بررسی آپدیت» آخرین Release را از GitHub بررسی می‌کند. نصب خودکار ZIP از داخل افزونه توسط Chrome مجاز نیست؛ برای انتشار عمومی از Chrome Web Store استفاده کنید.

## فونت
