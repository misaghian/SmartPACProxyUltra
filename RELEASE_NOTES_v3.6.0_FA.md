# یادداشت انتشار Smart PAC Ultra v3.6.0

## تمرکز نسخه

بازطراحی Popup با مدل «اقدام‌محور» برای اینکه کارهای اصلی همیشه دم دست باشند.

## تغییرات اصلی

- افزودن سایت فعلی به «مستقیم» یا «پروکسی» در صفحه اصلی Popup قرار گرفت.
- بخش ابزارها دیگر زیر گزارش کوتاه مخفی یا فشرده نمی‌شود؛ ابزارها اول نمایش داده می‌شوند و گزارش پایین‌تر قرار دارد.
- UI فشرده‌تر شد و ارتفاع کارت‌ها کاهش پیدا کرد.
- لیست مستقیم و پروکسی در صفحه اصلی قابل مشاهده و ذخیره سریع شد.
- فونت از مسیر `ui/fonts/` و مسیر جایگزین `assets/ui/fonts/` خوانده می‌شود.
- تنظیم سریع پروکسی، تست اتصال و نتیجه تست در صفحه اصلی باقی ماند.
- نسخه‌بندی به 3.6.0 ارتقا یافت.

## نکته آپدیت

Storage Key تغییر نکرده است؛ بنابراین لیست‌ها، پروفایل‌ها و تنظیمات کاربر در آپدیت حفظ می‌شوند.
