# یادداشت انتشار v3.4.2

این نسخه یک Patch مهم برای منطق PAC است.

## اصلاح اصلی

در نسخه‌های قبلی اگر یک IP عمومی را داخل لیست مستقیم وارد می‌کردید، PAC قبل از بررسی لیست مستقیم، IP را به مسیر پیش‌فرض می‌فرستاد. در این نسخه ترتیب بررسی قوانین اصلاح شد.

## پشتیبانی قوانین IP

اکنون در لیست مستقیم یا لیست پروکسی می‌توانید این موارد را وارد کنید:

```text
8.8.8.8
1.2.3.0/24
10.10.0.0/255.255.0.0
5.5.5.10-5.5.5.20
192.168.*
```

## نتیجه

اگر IP را در لیست مستقیم قرار دهید، همان IP بدون پروکسی باز می‌شود. اگر IP را در لیست پروکسی قرار دهید، با پروکسی باز می‌شود. قوانین صریح کاربر از تصمیم پیش‌فرض PAC اولویت بالاتری دارند.
