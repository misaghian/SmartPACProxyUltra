# نصب فونت VazirFD

CSS افزونه از فونت VazirFD پشتیبانی می‌کند. برای استفاده در نسخه محلی یا نسخه انتشار، فایل‌های فونت را در همین مسیر کنار `ui.css` قرار دهید:

```text
assets/ui/VazirFD.woff2
assets/ui/VazirFD.woff
assets/ui/VazirFD.ttf
```

در این بسته فایل فونت قرار داده نشده است. بعد از قرار دادن فایل‌ها، نیازی به تغییر CSS نیست.
