# Chrome Web Store Listing - FA

## عنوان
Smart PAC Ultra - مدیریت هوشمند پروکسی و PAC

## توضیح کوتاه
مدیریت ساده و حرفه‌ای پروکسی، PAC و حالت هوشمند با رابط فارسی.

## توضیح کامل
Smart PAC Ultra یک افزونه فارسی برای مدیریت سریع حالت‌های اتصال مرورگر است. کاربران می‌توانند بین Direct، Proxy، PAC، System Proxy و Smart Mode جابه‌جا شوند، پروفایل‌های مختلف پروکسی بسازند و برای هر پروفایل از لیست مشترک یا اختصاصی استفاده کنند.

## مجوزها
- proxy: اعمال تنظیمات اتصال مرورگر
- storage: ذخیره محلی تنظیمات
- tabs / activeTab: تشخیص دامنه تب فعلی برای افزودن سریع
- webNavigation: تشخیص خطاهای باز نشدن سایت
- webRequest: مشاهده درخواست‌ها برای وضعیت و تحلیل محلی
