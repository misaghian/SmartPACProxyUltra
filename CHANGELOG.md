# تغییرات Smart PAC Ultra

## v3.4.2

- رفع مشکل PAC برای IPهای واردشده در لیست مستقیم.
- تغییر ترتیب تصمیم‌گیری PAC: قوانین پروکسی و مستقیم قبل از تصمیم پیش‌فرض IP بررسی می‌شوند.
- پشتیبانی بهتر از قانون IP دقیق، wildcard مثل `192.168.*`، CIDR مثل `10.10.0.0/16` و بازه مثل `5.5.5.10-5.5.5.20`.
- اصلاح پاک‌سازی ورودی لیست‌ها تا CIDR با `/24` اشتباهی حذف نشود.
- حفظ تنظیمات قبلی کاربران با همان Storage Key نسخه‌های 3.4.x.


## v3.4.1

- اصلاح فوری UI Popup و صفحه کامل.
- اصلاح مسیر فونت VazirFD برای `ui/fonts/` و `assets/ui/fonts/`.
- اصلاح اندازه و ظاهر آیکون‌های خطی.
- اصلاح نمایش نتیجه تست اتصال در Popup.
- جلوگیری از خروج متن از دکمه‌ها، مخصوصاً حالت‌های اتصال.
